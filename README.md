VoiceInsight is an AI-powered call analysis engine that converts audio calls
into text and extracts useful insights like sentiment and intent.

## Tech Stack
- Backend: FastAPI
- Speech-to-Text: Whisper
- NLP: BERT (planned)
- Frontend: Streamlit
- Language: Python

## Current Features
- Audio upload
- Speech-to-text transcription using Whisper

## How to Run
1. Activate virtual environment
2. Install requirements
3. Run FastAPI server

This project is being developed step-by-step as a learning-focused college project.
